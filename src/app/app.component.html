Introduction :

The cockpit server is an application for monitoring the execution of data migration jobs for extraction and loading. The cockpit server displays the job execution details, job status, percentage of completion and validation details. The server provides detailed information regarding the progress of the jobs for various PLM objects (Parts, Drawings, EBOM, MBOM, Documents, other types)
